# 🚀 React Projects Collection

This repository contains multiple React.js projects built while learning and practicing core concepts of React.

---

## 📂 Projects Included

### 1. 01_basic

* Introduction to React basics
* Components, JSX, props

### 2. 02_basic

* Advanced basic concepts
* State management and event handling

### 3. 03_hooks

* React Hooks (useState, useEffect)
* Functional components

### 4. 04_react_useeffect

* Deep dive into useEffect hook
* API calls and lifecycle behavior

### 5. currency_converter_custom_hook

* Custom React Hook implementation
* Real-time currency conversion using API

### 6. passwordGen_using_hooks

* Password generator app
* Uses hooks and clipboard functionality

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3

---

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/react-projects.git
```

### 2. Navigate to any project

```bash
cd 01_basic
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
npm start
```

---

## 📌 Notes

* Each folder is an independent React project
* Install dependencies separately for each project
* `node_modules` is not included in the repository
